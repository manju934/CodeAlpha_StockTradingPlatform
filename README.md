# CodeAlpha Stock Trading Platform

A command-line Core Java engine built to simulate stock exchanges, manage catalogs, evaluate investor net-worth metrics, and run price fluctuation simulations.

## Features
1. **Stock Registry**: Add company indices, adjust active values, and remove listings.
2. **Investor Profiles**: Track investor holdings and cash balances.
3. **Trading System**: Run transactional checks for buying/selling stocks with audit history.
4. **Market Simulation**: Generate microeconomic shifts (-10% to +10%) across active assets.
5. **Sorting & Analytics**: Track portfolio growth metrics and sort tables.
6. **Serialization Layer**: Standard serialization persistence ensures persistence.

## Prerequisites
* Eclipse IDE.
* JDK 17 or higher.
